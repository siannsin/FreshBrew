# FreshBrew

FreshBrew is a macOS menu bar utility for checking and updating Homebrew formulae and casks.

The project is currently in early development. Its initial build targets Apple Silicon Macs running macOS 14 or later.

## Development

1. Open `FreshBrew.xcodeproj` in Xcode.
2. Select the `FreshBrew` scheme and `My Mac` destination.
3. Build and run with `Command-R`.

The development build uses the generic macOS application icon and built-in SF Symbols. Original FreshBrew artwork will be added before a public release.
